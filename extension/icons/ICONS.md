## Icon Requirements

Need PNG icons in these sizes:
- icon16.png (16x16) — toolbar, favicon
- icon32.png (32x32) — Windows taskbar
- icon48.png (48x48) — extensions page
- icon128.png (128x128) — Chrome Web Store, install dialog

### Design Brief

- Simple, recognizable at 16px
- Concept: a floppy disk / save icon with a small image/photo symbol overlay
- Or: a downward arrow into a frame (representing image download)
- Colors: clean blue (#4A90D9) primary, white background
- Style: flat/material design, no gradients
- No text in the icon (unreadable at 16px)

### Generate with

Use any image generator or manually create SVG and export to PNG sizes.
